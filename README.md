# Meta VR Setup (SDK v77)

Base Unity project configured for **Meta Quest VR development** using **Meta SDK v77**.  
Designed as a clean starting point for building VR experiences on **Unity 6 and above**.

## Features
- Meta SDK v77 preconfigured  
- XR Management and Oculus Loader ready  
- Input system set up (controllers + hands)  
- Sample scene for quick testing  
- Optimized project settings for Quest builds  

## Requirements
- **Unity 6 or newer**  
- **Meta Quest / Quest 3 headset**  
- **Meta SDK v77**

## Getting Started
1. Clone this repository  
2. Open in Unity  
3. Connect your Quest via USB  
4. Press **Play** or build to device
